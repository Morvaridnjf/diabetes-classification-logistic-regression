# Diabetes Classification with Logistic Regression

## Goal
This project builds a simple classification model to predict diabetes status using clinical variables.

## Dataset
The dataset contains patient-level features such as glucose, BMI, age, insulin, and diabetes outcome.

## Methods
- Data exploration
- Train/test split
- Feature scaling
- Logistic regression
- Regularization experiment
- Evaluation using accuracy, precision, recall, F1-score, and AUC

## Key Learning
This project helped me understand how logistic regression is used for binary classification and how regularization affects model performance.

## Clinical Interpretation
In a medical context, recall/sensitivity is especially important because false negatives may represent missed diabetic patients.
